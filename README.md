# ABOUT THE WEBSITE

An inventory Management website to show off simple API fetching and CRUD functionality with client-side data.
What you can do in the website:  

View products from the fakestoreapi  
Add products  
Delete products  
Edit products  
Reset the page  

Future implements:  

Customer View  
Customizable Store Name
